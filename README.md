# Isaac Rebith on R36S.

![](https://raw.githubusercontent.com/vanyat05/Isaac-Rebith-on-R36S./main/logo.jpg)
Данная информация поможет владельцам консоли R36S установить игру "Isaac Rebith"  на прошивку Arkos.
